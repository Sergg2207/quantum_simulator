# Simulador de Sistemas Cuanticos

Simulador para sistemas cuanticos discretos (basado en el libro Quantum Computing for Computer Scientists)

## Caracteristicas

-  Sistema de particula en posiciones discretas
-  Calculo de probabilidades de posición
-  Amplitudes de transicion entre estados
- Observables hermitianos y mediciones
-  Dinamica con matrices unitarias
-  Ejemplos y problemas del capítulo 4
-  Tests unitarios

## Instalación Rápida

```bash
# Clonar el repositorio
git clone https://github.com/Sergg2207/quantum_simulator
cd quantum-simulator

# Instalar dependencias
pip install -r requirements.txt
